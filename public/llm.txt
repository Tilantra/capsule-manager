# Capsule Hub by Tilantra
# https://capsulehub.tilantra.com
# Last updated: 2026-06-07

> Capsule Hub is a context-transfer layer for modern AI work. It turns chats, email threads, and custom content into portable Capsules—structured bundles of goals, constraints, background, and attachments—so you can move work between models and tools without rebuilding context from scratch.

## What is Capsule Hub?

Capsule Hub by Tilantra is the bridge for your AI workflows. Most teams lose hours re-explaining the same situation to every new chat. Capsule Hub makes that knowledge capture-once, inject-anywhere: from research in Gemini to execution in ChatGPT, from a Gmail thread to requirements in GPT, from a capsule in Cursor (via MCP) back to review in Claude—with versioning along the way.

A Capsule is a structured context packet containing conversation messages, metadata, and optional file attachments. Capsules can be captured via the Chrome extension, created manually in the web studio, managed through MCP in IDEs, or embedded via the Personal Chatbot SDK.

## Core Features

- Chrome Extension Capture: One-click capture from ChatGPT, Claude, Google Gemini, Perplexity, DeepSeek, Gmail, and more
- Drag-and-Drop Injection: Drag capsules from your library into any supported AI chat input for instant context injection
- Context Versioning: Preserve history, roll back, compare versions, and tag stable checkpoints
- Merge & Split Capsules: Combine multiple capsules into one unified context, or split a capsule into focused sub-capsules for role-based sharing
- Attachments: Rich file support (20+ types) including PDFs, Markdown, text, code, and images
- Teams & Collaboration: Organize capsules by department, share with team members, and collaborate on evolving prompts
- Dynamic Context (Elite): AI-powered semantic filtering to keep injected payloads relevant as conversations evolve
- MCP Integration: Connect Capsule Hub to Cursor, Antigravity, and other MCP-compatible IDEs
- Personal Chatbot SDK: Embed Capsule Hub context on your own website chatbot
- Anthropic Skills: Use Claude Code skills to save, search, read, and version capsules from terminal
- Custom Capsule Studio: Build capsules directly in a connected graph canvas on the web
- Multi-Model Support: Move context seamlessly between ChatGPT, Claude, Gemini, Perplexity, DeepSeek, Copilot, and more
- Zero Context Rot: Prevent hallucinations and context loss when switching between AI tools
